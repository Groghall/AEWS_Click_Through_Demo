About this page

This is the log-in page for CSFS Auto-Enrol Workplace Solution (AEWS), a secure web-based service designed to help employers manage their auto-enrolment obligations.

The company will receive their user account details from CSFS AE Support Team. Once they have this, the company can log into AEWS for the first time. AEWS will prompt the user to change the password, provide information for three security questions and security information to be held on the system too enable password resets.

Security features

  • Perimeter protection for all traffic entering and or leaving Workplace Solution with dedicated global DNS servers.
  • Resilient multi-protocol label switching (MPLS) network, using V*Net.
  • Virtual multi-tenanted modules, each with its own configurable security elements.
  • Secure virtual storage area network, backed by electronic vaulting with the back-up data held and available at separate data centres.
